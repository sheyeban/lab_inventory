# Lab Inventory (Тема 5)

Консольное приложение на **Rust** для учёта лабораторного оборудования с базой данных **SQLite**, ролями пользователей и журналом действий.

## Возможности

* учёт оборудования (InventoryItem);
* категории и поставщики (Category, Vendor);
* история обслуживания (MaintenanceLog);
* роли пользователей: **LabManager / Assistant**;
* хеширование паролей (**bcrypt**);
* журнал действий (**AuditLog**);
* поиск и фильтрация данных.

## Технологии

* Rust
* SQLite (rusqlite)
* bcrypt
* chrono

## Роли

**LabManager** — полный доступ (CRUD, пользователи, audit log).
**Assistant** — просмотр, изменение статуса, обслуживание.

## Запуск

```bash
cargo run
```
